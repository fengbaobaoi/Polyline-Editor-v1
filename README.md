# Polyline-Editor-v1
一款交互式 Excel 折线图编辑工具，支持上传 Excel/CSV 文件，自定义选择工作表、X 轴及多列 Y 轴数据生成折线图。图表支持鼠标滚轮缩放、拖拽平移，数据点可拖拽修改、双击删除，折线可单击隐藏 / 显示。修改后的数据可实时同步至表格预览，支持一键重置数据 / 视图、导出新 Excel 文件。
一、详细功能说明
1.核心功能
文件导入：支持上传.xlsx/.xls/.csv 格式文件，自动识别工作表并解析数据；
图表配置：自定义选择数据源工作表、X 轴数据列、多列 Y 轴数据列，灵活生成折线图；
2.图表交互
缩放：鼠标滚轮放大 / 缩小图表（同步缩放 X/Y 轴），支持重置视图；
编辑：拖拽数据点修改数值，双击数据点删除数据，单击图例隐藏 / 显示对应折线；
平移：通过滚动条横向平移图表；
数据同步：图表修改后的数据实时同步至右侧数据预览表格；
数据管理：一键重置修改后的数据至原始状态；
导出功能：将编辑后的数据导出为新的 Excel 文件；
适配性：支持移动端 / 桌面端响应式布局，保证不同设备操作体验。
3.辅助功能
自动检测数据精度，tooltip 精准显示数值；
上传 / 操作过程实时状态提示（成功 / 错误 / 加载）；
数据表格支持滚动预览，表头固定，行高亮交互；
全局错误捕获，友好提示异常信息。
